## 项目简介

**人从众族 (zu)** 是一款基于 Flutter 开发的数字化亲情族谱应用，旨在帮助用户建立和维护家族关系网络，实现亲情互认。

## 核心功能

### 族谱数字化
- 可视化家族树结构
- 支持多代亲属关系记录
- 家族成员信息管理

### 亲情互认
- 家族成员身份验证
- 亲属关系确认机制
- 家族动态分享

## 技术栈

| 技术 | 用途 |
|------|------|
| Flutter | 跨平台移动应用开发 |
| Riverpod | 状态管理 |
| Go Router | 路由导航 |
| Firebase | 后端服务（认证、消息、分析） |
| Hive | 本地数据存储 |
| Dio | 网络请求 |

## 项目结构

```
zu/
├── zu_app/        # Flutter 应用端
├── zu_server/     # 服务端
└── 资料/          # 项目资料
```

## 特色亮点

1. **跨平台支持** - 一套代码，同时支持 iOS 和 Android
2. **Firebase 集成** - 完整的后端服务支持
3. **离线优先** - Hive 本地存储，支持离线使用
4. **精美动画** - flutter_animate 提供流畅的交互体验
5. **二维码分享** - 支持 QR 码快速添加家族成员
